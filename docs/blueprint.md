# **App Name**: VeggiePal

## Core Features:

- Meal Tracker: Input meals, track daily nutritional intake.
- Intelligent Nutrition Lookup: Use the meal name, ingredients, and portion to look up accurate nutrition information via a 3rd-party nutrition API. Act as a tool: estimate missing information, and offer multiple suggestions from the nutrition API when the entered data yields ambiguous results. Present results with most likely/most relevant option first.
- Nutritional Summary: Display daily nutritional totals including calories, macros, and micronutrients.
- Goal Setting: Set personalized dietary goals (calories, macros, micros) for weight loss, gain, or maintenance.
- Recipe Suggestions: Suggest vegan-friendly recipes based on user preferences, available ingredients, and nutritional goals, using a 3rd-party recipe API or LLM.
- History: Maintain history of the daily nutrition intake to present intake information across various different days.

## Style Guidelines:

- Primary color: Fresh green (#8FBC8F) to evoke health and nature.
- Background color: Light green (#F0FFF0), nearly white, to provide a clean backdrop.
- Accent color: Earthy brown (#A0522D) to complement the natural theme.
- Body and headline font: 'Alegreya', a humanist serif with a contemporary feel.
- Code font: 'Source Code Pro' for displaying any code snippets or nutritional data.
- Use minimalist icons representing different food groups and nutrients.
- Clean, spacious layout with a focus on readability and easy navigation.
